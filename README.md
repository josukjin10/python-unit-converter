# Multi-Tool Python Units Converter

A robust, terminal-based application written in Python that allows users to easily convert units across five major categories. Built with user-experience in mind, the program features dynamic menus, strict input validation to prevent crashes, and a continuous loop system.

## Features

- **5 Conversion Categories:** Distance/Length, Temperature, Mass, Volume, and Time.
- **Continuous Loop:** Run as many conversions as you want without the program closing, thanks to an integrated `while` loop workflow.
- **Crash-Proof Design:** Robust input validation catches typos or invalid selections gracefully rather than letting the script crash.
- **Pivot-Unit Efficiency:** Uses centralized math arrays to keep code efficient and highly accurate.

## How It Works

1. Run the notebook in Google Colab or any Python environment.
2. Select a measurement type from the main menu (1-5).
3. Choose your starting unit and the target unit.
4. Input the numerical value you wish to convert.
5. View your rounded result instantly, and choose to continue or type `exit` to quit!

## Tech Used

- **Language:** Python 3
- **Environment:** Google Colab / Jupyter Notebooks
- **Concepts Applied:** Nested conditional logic (`if/elif/else`), arrays/lists, input validation, and `while` loops.
